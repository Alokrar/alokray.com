# alokray.com

This repository contains a minimal static homepage for alokray.com. Files are served from the `gh-pages` branch.

Files included:
- `index.html` — main page
- `CNAME` — custom domain (alokray.com)
- `.nojekyll` — disables Jekyll processing

To edit the site, modify `index.html` and push to the `gh-pages` branch. If you need a site generator or CI build, open an issue or request changes.
